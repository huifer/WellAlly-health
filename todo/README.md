## 按人群分类

### 👩 女性健康

**文件**: [01-womens-health.md](01-womens-health.md)

**子模块**:
- 🤰 孕期管理系统
- 🌸 更年期管理系统
- 🎗️ 妇科癌症筛查追踪

**核心功能**:
- 预产期计算、孕周追踪
- 产检计划提醒（NT、唐筛、糖耐、大排畸）
- 孕期症状记录（孕吐、水肿、胎动）
- 孕期用药安全检查（妊娠分级A/B/C/D/X）
- 更年期症状评分（潮热、情绪波动）
- HRT治疗记录和风险监测
- HPV/TCT筛查管理
- 妇科肿瘤标志物追踪（CA125、CA19-9）

## 贡献指南

### 添加新功能

1. 创建新的markdown文件，命名格式：`编号-功能名称.md`
2. 在文件中包含以下部分：
   - 功能描述
   - 核心功能
   - 数据结构
   - 命令接口
   - 医学安全原则

3. 更新本README.md，将新功能添加到目录中

### 更新现有功能

1. 直接编辑对应的markdown文件
2. 更新功能状态和优先级
3. 在文件末尾添加更新日志

### 状态标记

- 📝 待开发 - 功能提案阶段
- 🔨 开发中 - 正在实现
- ✅ 已完成 - 功能已实现
- ⏸️ 暂缓 - 暂时搁置
- ❌ 已取消 - 不再计划实现

## 文档规范

每个功能提案文档应包含以下章节：

1. **功能描述** - 简要说明功能的目的和价值
2. **核心功能** - 列出主要功能点
3. **数据结构** - JSON格式的数据模型
4. **命令接口** - CLI命令示例
5. **使用示例** - 典型使用场景
6. **医学安全原则** - 相关安全限制
7. **注意事项** - 特殊说明
8. **参考资源** - 相关文档链接
