# 预防保健协调员技能设计

## 概述
**技能名称**: `preventive-care-coordinator`
**目的**: 整合预防保健措施，包括疫苗接种、健康筛查、体检计划等。

## 描述
基于年龄、性别、风险因素和家族史，提供个性化的预防保健计划，包括疫苗接种时间表、筛查检查安排、健康风险评估和预防建议。在需要预防保健规划、疫苗接种提醒或询问"我该做哪些体检？"时使用。

## 数据集成

### 数据源
- **个人档案** (`data/profile.json`): 年龄、性别、基础信息
- **疫苗接种** (`data/vaccines/`): 接种记录
- **家族病史**: 家族疾病史
- **手术记录** (`data/手术记录/`): 手术史
- **化验报告** (`data/medical-reports/`): 筛查结果
- **生活方式**: 风险因素评估

### 相关命令
- `/vaccine`: 疫苗接种
- `/query`: 数据查询
- `/report`: 健康报告

## 核心功能

### 1. 疫苗接种管理
- **接种时间表**: 基于年龄和风险
- **疫苗接种记录**: 已接种疫苗
- **疫苗提醒**: 即将到期的疫苗
- **疫苗禁忌**: 接种禁忌评估
- **特殊人群**: 孕期、慢性病等

### 2. 健康筛查计划
- **年龄appropriate筛查**: 基于年龄的筛查
- **风险评估**: 基于风险因素的筛查
- **筛查时间表**: 何时做什么检查
- **结果追踪**: 追踪筛查结果
- **异常管理**: 异常结果的随访

### 3. 健康体检规划
- **体检项目**: 推荐体检项目
- **体检频率**: 多久体检一次
- **体检机构**: 去哪里做检查
- **体检准备**: 检查前准备
- **结果解读**: 结果含义

### 4. 风险评估
- **疾病风险**: 慢性病风险评估
- **生活方式风险**: 可改变的风险因素
- **家族史风险**: 遗传风险评估
- **环境风险**: 环境暴露风险
- **职业风险**: 工作相关风险

```
□ 大量流质（水、清汤、果汁）
□ 避免: 固体食物、红色食物、酒精

□ 清肠药物（按说明）
□ 持续液体摄入
□ 不吃固体食物
□ 不喝水（检查前 4 小时）

检查后:
□ 可能嗜睡（镇静）
□ 需要有人陪同
□ 当天休息
□ 次日恢复工作
□ 检查后腹胀正常

血脂检查准备:
□ 空腹 12 小时
□ 前一天晚餐清淡
□ 避免饮酒 24-48 小时
□ 避免高脂饮食 24 小时
□ 可喝水

血糖/A1C:
□ 空腹 8 小时
□ 可喝水
□ 避免过度运动前晚

□ 充足睡眠

□ 避免过度运动
```

### 预防保健计划生成

```python
def generate_preventive_plan(profile, history, risk_factors):
    """生成个性化预防保健计划"""

    # 基于年龄和性别的标准筛查
    standard_screening = get_standard_screening(
        profile["age"],
        profile["gender"]
    )

    # 基于风险的额外筛查
    risk_based = get_risk_based_screening(
        risk_factors,
        history
    )

    # 疫苗接种时间表
    vaccines = get_vaccine_schedule(
        profile["age"],
        profile["conditions"],
        risk_factors
    )

    # 优先级排序
    prioritized = prioritize_screenings(
        standard_screening,
        risk_based,
        profile
    )

    # 生成时间表
    schedule = create_annual_schedule(
        prioritized,
        vaccines
    )

    return {
        "screening": prioritized,
        "vaccines": vaccines,
        "schedule": schedule,
        "action_items": get_immediate_actions(prioritized),
        "cost_estimate": estimate_costs(schedule)
    }
```
