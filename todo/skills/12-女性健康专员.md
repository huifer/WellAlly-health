# 女性健康专员技能设计

## 概述
**技能名称**: `women-health-specialist`
**目的**: 整合女性全生命周期健康管理，包括月经、孕期、更年期等各个阶段。

## 描述
整合女性周期追踪、孕期管理、更年期管理等全生命周期健康数据，提供综合女性健康分析和建议。在需要女性健康评估、周期问题、孕期追踪或更年期管理时使用。

## 数据集成

### 数据源
- **周期记录** (`data/cycle/`): 月经周期数据
- **孕期记录** (`data/pregnancy/`): 孕期追踪数据
- **更年期记录** (`data/menopause/`): 更年期症状
- **症状记录** (`data/symptoms/`): 女性健康症状
- **情绪记录** (`data/mood/`): 激素相关情绪变化
- **药物记录** (`data/medications/`): 激素治疗等

### 相关命令
- `/cycle`: 女性周期管理
- `/pregnancy`: 孕期管理
- `/menopause`: 更年期管理
- `/symptom`: 症状追踪
- `/mood`: 情绪记录

## 核心功能

### 1. 生命周期整合分析
- **阶段识别**: 识别当前所处生命周期阶段
- **激素变化**: 追踪激素相关变化
- **跨阶段趋势**: 分析长期健康趋势
- **风险评估**: 不同阶段特定风险
- **预防保健**: 基于年龄和阶段的筛查

### 2. 周期健康
- **周期规律性**: 评估周期是否规律
- **症状追踪**: 经前症状、痛经等
- **排卵监测**: 排卵期识别
- **生育能力**: 基于周期的生育评估
- **异常识别**: 异常出血、周期紊乱

### 3. 孕期管理
- **孕期进度**: 追踪孕周和胎儿发育
- **症状管理**: 孕期常见症状
- **营养需求**: 孕期营养指导
- **运动建议**: 孕期appropriate运动
- **警示征象**: 何时就医

### 4. 更年期管理
- **阶段评估**: 围绝经期阶段
- **症状管理**: 潮热、情绪等
- **激素治疗**: HRT 考虑
- **骨密度**: 骨骼健康
- **心血管风险**: 心脏健康评估

### 女性生命周期评估
```python
def assess_womens_health(life_stage, patient_data):
    """评估女性健康"""

    if life_stage == "reproductive":
        return assess_reproductive_health(patient_data)
    elif life_stage == "perimenopause":
        return assess_perimenopause(patient_data)
    elif life_stage == "postmenopause":
        return assess_postmenopause(patient_data)
    elif life_stage == "pregnancy":
        return assess_pregnancy(patient_data)

def assess_perimenopause(data):
    """围绝经期评估"""
    cycle_data = data["cycle_history"]
    symptoms = data["symptoms"]

    # 判断阶段
    stage = determine_perimenopause_stage(cycle_data)

    # 风险评估
    risks = {
        "bone": assess_bone_risk(data),
        "cardiovascular": assess_cardio_risk(data),
        "breast_cancer": assess_breast_cancer_risk(data)
    }

    # 管理建议
    recommendations = generate_recommendations(stage, symptoms, risks)

    return {
        "stage": stage,
        "symptoms": analyze_symptoms(symptoms),
        "risks": risks,
        "screening": create_screening_plan(data),
        "management": recommendations
    }
```
