# 慢性病管理教练技能设计

## 概述
**技能名称**: `chronic-disease-coach`
**目的**: 为慢性病患者提供长期疾病管理、监测和指导，包括高血压、糖尿病、心脏病等。

## 描述
针对慢性病（高血压、糖尿病、心脏病、慢阻肺等）提供持续的健康管理、目标监控、生活方式调整和并发症预防。在需要管理慢性病、控制病情或询问"我的血压/血糖控制得怎么样？"时使用。

## 数据集成

### 数据源
- **药物记录** (`data/medications/`): 慢性病用药管理
- **症状记录** (`data/symptoms/`): 疾病相关症状
- **个人档案** (`data/profile.json`): 基础健康指标
- **饮食记录** (`data/diet/`): 饮食对慢性病的影响
- **运动记录**: 运动对疾病管理的影响
- **化验报告** (`data/medical-reports/`): 关键指标追踪
- **生命体征**: 血压、血糖、体重等监测数据

### 相关命令
- `/medication`: 慢性病药物管理
- `/symptom`: 症状追踪
- `/diet`: 饮食管理
- `/query`: 数据查询

## 核心功能

### 1. 疾病状态监控
- **目标值追踪**: 血压、血糖、血脂等关键指标
- **趋势分析**: 长期控制趋势
- **达标评估**: 是否达到治疗目标
- **波动识别**: 识别异常波动
- **并发症预警**: 早期并发症迹象

### 2. 用药管理
- **依从性追踪**: 是否按时服药
- **药物效果评估**: 症状和指标改善
- **副作用监测**: 药物副作用识别
- **用药调整建议**: 基于效果调整
- **药物相互作用**: 多重用药管理

### 3. 生活方式干预
- **饮食调整**: 针对疾病的饮食建议
- **运动处方**: 疾病appropriate运动
- **体重管理**: 对疾病的正面影响
- **戒烟戒酒**: 生活方式改变
- **压力管理**: 压力对疾病的影响

### 4. 教育与赋能
- **疾病知识**: 理解自己的疾病
- **自我监测技能**: 在家监测技能
- **警示征象**: 何时寻求帮助
- **预防措施**: 预防急性发作
- **生活质量**: 带病生活优化

📚 患者教育

高血压基础知识:
□ 血压定义: 心脏泵血对血管壁的压力
□ 正常血压: <120/80 mmHg
□ 高血压诊断: ≥140/90 mmHg（重复测量）
□ 靶器官损害: 心脏、肾脏、脑、血管、眼睛

家庭监测正确方法:
✓ 安静环境休息 5 分钟
✓ 坐位，背部支撑，双脚平放
✓ 手臂与心脏同高
✓ 使用合适袖带尺寸
✓ 测量前 30 分钟无咖啡因、运动
✓ 同一时间、同侧手臂、同一姿势
✓ 记录日期、时间、数值

何时紧急就医:
⚠️ 收缩压 ≥180 或 舒张压 ≥120
⚠️ 伴有: 胸痛、呼吸困难、头痛、头晕、视力模糊
⚠️ 疑似高血压急症

平时就医:
□ 血压持续高于目标 2 周
□ 药物副作用无法耐受
□ 新发症状
□ 定期随访（每 3-6 个月）

### 慢性病评估算法
```python
def assess_chronic_disease(disease_type, patient_data):
    """评估慢性病控制状态"""

    if disease_type == "hypertension":
        return assess_hypertension(patient_data)
    elif disease_type == "diabetes":
        return assess_diabetes(patient_data)
    elif disease_type == "heart_disease":
        return assess_heart_disease(patient_data)
    # ... 其他疾病

def assess_hypertension(data):
    """高血压评估"""
    bp_data = data["blood_pressure_readings"]
    targets = {"sbp": 130, "dbp": 80}

    assessment = {
        "current_bp": calculate_average(bp_data),
        "target_met": check_targets(bp_data, targets),
        "trend": analyze_trend(bp_data),
        "variability": calculate_variability(bp_data),
        "control_quality": evaluate_control(bp_data, targets)
    }

    # 并发症风险
    risk_factors = identify_risk_factors(data)
    complication_risk = calculate_complication_risk(assessment, risk_factors)

    return {
        "assessment": assessment,
        "medication": evaluate_medication(data),
        "lifestyle": evaluate_lifestyle(data),
        "education": generate_education("hypertension"),
        "followup": create_followup_plan(assessment)
    }
```

- 指标持续不达标
- 新发症状
