# 康复指导师技能设计

## 概述
**技能名称**: `rehabilitation-guide`
**目的**: 提供手术或疾病后的康复指导、追踪和管理。

## 描述
为手术后、创伤后、疾病康复期的患者提供个性化康复计划、进度追踪和指导。整合手术记录、症状管理、运动处方等，提供全面的康复支持。在需要康复指导、术后管理或询问"手术后如何康复？"时使用。

## 数据集成

### 数据源
- **手术记录** (`data/手术记录/`): 手术类型、日期
- **症状记录** (`data/symptoms/`): 疼痛、功能障碍
- **药物记录** (`data/medications/`): 疼痛管理
- **运动记录**: 康复运动进展
- **个人档案** (`data/profile.json`): 年龄、基础健康状况

### 相关命令
- `/surgery`: 手术记录
- `/symptom`: 症状追踪
- `/medication`: 疼痛管理

## 核心功能

### 1. 术后康复计划
- **手术类型识别**: 不同手术类型
- **康复时间表**: 分期康复计划
- **活动指导**: 逐步恢复活动
- **限制事项**: 需要避免的活动
- **里程碑**: 康复里程碑

### 2. 疼痛管理
- **疼痛评估**: 疼痛类型和强度
- **药物管理**: 镇痛药使用
- **非药物方法**: 冰敷、抬高、休息
- **疼痛教育**: 正常疼痛 vs 警示信号
- **减药策略**: 逐渐减少镇痛药

### 3. 功能恢复
- **关节活动度**: ROM 练习
- **力量训练**: 渐进式力量恢复
- **日常生活能力**: ADL 恢复
- **功能评估**: 定期功能测试
- **回归活动**: 运动、工作、驾驶

### 4. 并发症监测
- **感染迹象**: 伤口感染
- **血栓预防**: DVT 征象
- **其他并发症**: 特定手术并发症
- **警示征象**: 何时就医
- **预防措施**: 预防性措施

💡 康复提示

成功关键:
✓ 重复练习 > 频率
✓ 质量 > 数量
✓ 坚持是关键
✓ 耐心（康复需要时间）
✓ 倾听身体（适度疼痛可接受）
✓ 逐渐增加（不要过度）

常见错误:
✗ 进步太快
✗ 忽略疼痛
✗ 跳过练习
✗ 不做ROM练习
✗ 强力活动过早
✗ 不遵循限制

康复里程碑:
✓ 伸膝正常 → 可更多练习
✓ 步态正常 → 可减少助行
✓ 股四头肌力量 → 功能恢复
✓ 平衡正常 → 回归运动

何时联系医生:
□ 疼痛加重或持续
□ 肿胀不消退
□ ROM 无进展 2 周
□ 新症状出现
□ 伤口问题
□ 不确定某项练习

急诊情况:
→ 剧烈疼痛
→ 大量出血
→ 伤口裂开
→ 发热 >38°C
→ DVT 迹象
