# 健康教练技能设计

## 概述
**技能名称**: `wellness-coach`
**目的**: 基于综合健康数据分析，提供个性化健康指导、生活方式建议和行为教练。

## 描述
作为个人健康教练，基于个人健康数据提供生活方式建议、目标设定帮助、习惯养成指导和激励。在寻求健康建议、生活方式改变或询问"我该如何改善健康状况？"时使用。

## 数据集成

### 数据源
- **个人档案**: 当前健康状态
- **症状记录**: 生活方式如何影响症状
- **情绪记录**: 心理健康模式
- **饮食记录**: 营养习惯
- **睡眠记录**: 睡眠质量
- **运动记录**: 体力活动
- **健康目标**: 个人健康目标

### 相关技能
- 所有技能提供教练建议的数据
- `health-insights`: 提供教练基础

## 核心功能

### 1. 个性化建议
- **饮食与营养**: 膳食规划、营养指导
- **运动与健身**: 活动建议、锻炼计划
- **睡眠卫生**: 睡眠改善策略
- **压力管理**: 压力缓解技巧
- **生活方式优化**: 日常生活改善

### 2. 目标设定与追踪
- **SMART 目标**: 具体、可衡量、可达成、相关、有时限
- **目标分解**: 将大目标分解为可管理的步骤
- **进度追踪**: 监控目标达成
- **调整指导**: 基于进度修改目标

### 3. 习惯养成
- **习惯设计**: 创建可持续的健康习惯
- **触发点识别**: 理解习惯触发因素
- **习惯叠加**: 在现有习惯上建立新习惯
- **习惯追踪**: 监控习惯一致性

### 4. 激励与问责
- **进度认可**: 庆祝胜利和里程碑
- **鼓励**: 在挑战期间提供动力
- **问责检查**: 定期检查
- **障碍规划**: 预期并克服障碍

**技能**: 提供鼓励，提醒进度，重申承诺

## 教练理念

### 原则
1. **进度胜过完美**: 一致的小胜 > 偶尔的完美
2. **个性化**: 一刀切不适合所有人
3. **可持续性**: 建立持久的习惯，而非快速修复
4. **同理心**: 挫折是学习机会，而非失败
5. **基于证据**: 基于科学的建议
6. **整体方法**: 关注整个人，不仅仅是症状

### 行为科学技巧
- **习惯叠加**: 在现有习惯上附加新习惯
- **实施意图**: "当 X 发生时，我会做 Y"
- **渐进式超负荷**: 逐渐增加难度
- **社会支持**: 利用问责伙伴
- **奖励系统**: 庆祝里程碑和胜利
- **认知重构**: 重构消极思维
