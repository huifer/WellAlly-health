## 🎯 功能概述

WellAlly-Health 项目包含全面的儿童健康管理功能，覆盖 **0-18岁全年龄段**，提供从新生儿到青少年的健康数据追踪、发育评估、疾病管理等功能。

### 主要特点

- ✅ **全年龄覆盖**: 0-18岁儿童与青少年
- ✅ **科学标准**: 基于WHO、中国卫健委等权威标准
- ✅ **本地存储**: 所有数据本地保存，保护隐私
- ✅ **智能分析**: AI驱动的健康数据分析
- ✅ **安全可靠**: 严格遵守医学安全原则

## 🔧 核心功能模块

### 1. 生长曲线追踪 (`/growth`)

**目标年龄**: 0-18岁
**优先级**: ⭐⭐⭐⭐⭐

#### 核心功能
- 📊 身高/体重/BMI记录与追踪
- 📏 头围测量（0-3岁）
- 📈 WHO标准百分位计算
  - 年龄别身高 (HAZ)
  - 年龄别体重 (WAZ)
  - 身高别体重 (WHZ)
  - 年龄别BMI (BAZ)
- 📉 生长速度计算（cm/年，kg/年）
- ⚠️ 生长异常预警
  - 生长迟缓：身高 < -2SD
  - 体重不足：体重 < -2SD
  - 超重：体重 > +1SD
  - 肥胖：BMI > +2SD

#### 命令示例
```bash
/growth record height 112.5 weight 20.5   # 记录身高体重
/growth status                            # 查看生长评估
/growth chart                             # 查看生长曲线
/growth velocity                          # 查看生长速度
/growth check                             # 检查生长异常
```

#### 数据文件
- `data/growth-tracker.json` - 实际数据存储
- `data-example/growth-tracker.json` - 数据示例
- `data/who-growth-standards.json` - WHO生长标准数据库

### 2. 青春期发育评估 (`/puberty`)

**目标年龄**: 8-18岁
**优先级**: ⭐⭐⭐⭐⭐

#### 核心功能
- 👦 **男孩发育评估**
  - 睾丸体积测量（Prader睾丸计）
  - 阴茎发育评估
  - 阴毛发育分期（P1-P5）
  - 变声、腋毛、胡须记录
  - Tanner分期（G1-G5）

- 👧 **女孩发育评估**
  - 乳房发育分期（B1-B5）
  - 阴毛发育分期（P1-P5）
  - 初潮年龄记录
  - 月经周期追踪

- 🦴 **骨龄评估**
  - 骨龄与实际年龄对比
  - 性早熟筛查
  - 发育延迟识别

#### 命令示例
```bash
# 女孩评估
/puberty breast B3              # 记录乳房发育
/puberty menarche true 10.2     # 记录初潮

# 男孩评估
/puberty testicular 8 ml        # 记录睾丸体积
/puberty voice true             # 记录变声

# 骨龄与状态
/puberty bone-age 10.8          # 记录骨龄
/puberty status                 # 查看发育状态
```

### 3. 儿童疫苗接种助手 (`/child-vaccine`)

**目标年龄**: 0-6岁（延伸至18岁）
**优先级**: ⭐⭐⭐⭐⭐

#### 核心功能
- 💉 **一类疫苗管理**（国家免疫规划）
  - 乙肝疫苗（0、1、6月龄）
  - 卡介苗（出生）
  - 脊灰疫苗（2、3、4、4岁）
  - 百白破疫苗（3、4、5、18月龄、6岁）
  - 麻腮风疫苗（8、18月龄）
  - 流脑疫苗（6、9月龄、3、6岁）
  - 乙脑疫苗（8月龄、2岁）
  - 甲肝疫苗（18月龄）

- 💊 **二类疫苗管理**（自费可选）
  - 水痘疫苗
  - 13价肺炎疫苗（PCV13）
  - 五联疫苗
  - 轮状病毒疫苗
  - 流感疫苗
  - EV71手足口疫苗

- 🔔 **智能提醒**
  - 即将到期提醒（提前7天）
  - 逾期未种预警
  - 接种间隔提醒

- 📝 **不良反应记录**
  - 发热程度记录
  - 局部反应记录
  - 过敏反应追踪

#### 命令示例
```bash
/child-vaccine record 乙肝疫苗 第1针 completed 2020-01-01
/child-vaccine schedule          # 查看接种计划
/child-vaccine upcoming          # 查看即将接种
/child-vaccine overdue           # 查看逾期疫苗
/child-vaccine reaction 百白破 发热 轻度
```

#### 数据文件
- `data/child-vaccinations.json` - 接种记录
- `data-example/child-vaccinations.json` - 数据示例
- `data/child-vaccine-database.json` - 疫苗数据库

## 🔬 技术特点

### 1. 标准化数据

- **WHO儿童生长标准**：0-5岁国际标准
- **中国儿童生长参照**：0-18岁本土数据
- **国家免疫规划**：最新疫苗接种程序
- **发育量表**：ASQ-3、Denver II等权威工具

### 2. 智能计算

- **Z-score计算**：标准差评分
- **百分位自动计算**：P3, P10, P25, P50, P75, P90, P97
- **生长速度**：基于历史数据的速度曲线
- **骨龄差异**：骨龄与实际年龄对比

### 3. 预警系统

- **生长异常**：自动识别生长迟缓、超重、肥胖
- **发育延迟**：里程碑延迟预警
- **性早熟筛查**：年龄-发育匹配检测
- **疫苗漏种**：自动提醒逾期未种

### 4. 数据可视化

- 文本格式生长曲线
- 百分位曲线图
- 疫苗接种时间轴
- 发育里程碑进度图

## ⚠️ 医学安全原则

### 安全红线（绝对禁止）

1. ❌ **不做医学诊断**
   - 仅提供数据记录和参考评估
   - 不替代医生专业诊断

2. ❌ **不推荐具体药物/品牌**
   - 不建议具体用药剂量
   - 不推荐药品品牌

3. ❌ **不处理紧急情况**
   - 高热、严重症状→引导就医
   - 系统仅记录，不指导处理

4. ❌ **不判断疫苗禁忌**
   - 禁忌症需医生判断
   - 系统仅提供计划管理

### 系统能做的（合规功能）

✅ 健康数据记录与追踪
✅ 基于标准的百分位计算
✅ 生长趋势可视化
✅ 异常预警（建议就医）
✅ 疫苗接种计划管理
✅ 发育里程碑记录
✅ 睡眠、营养、心理数据追踪

### 免责声明

```
⚠️ 重要提示：
本系统所有分析报告仅供健康记录和参考评估，
不能替代专业医疗诊断。
如有异常或疑虑，请及时咨询专业儿科医生。
紧急情况请立即就医或拨打120。
```

## 💡 使用建议

### 新生儿（0-1岁）
1. 首先设置儿童档案（姓名、出生日期、性别、出生体重/身高）
2. 定期记录生长数据（建议每月一次）
3. 按时记录疫苗接种
4. 追踪发育里程碑（重点：大运动、精细动作）
5. 记录睡眠模式和喂养情况

### 幼儿（1-3岁）
1. 继续追踪生长曲线
2. 完成疫苗补种
3. 重点关注语言和社交发育
4. 记录饮食习惯（挑食评估）
5. 建立规律作息

### 学龄前（3-6岁）
1. 生长监测（每3个月）
2. 心理健康筛查
3. 营养均衡评估
4. 安全教育记录
5. 入园体检数据管理

### 学龄期（6-12岁）
1. 生长曲线关注（每半年）
2. 营养和运动管理
3. 睡眠质量追踪
4. 心理健康关注
5. 疾病记录

### 青春期（12-18岁）
1. 青春期发育评估（重点）
2. 骨龄检测
3. 心理健康筛查（焦虑、抑郁）
4. 营养和体重管理
5. 睡眠问题识别
