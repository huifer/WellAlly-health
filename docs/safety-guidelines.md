# 安全原则和使用限制

## 专家会诊系统安全原则

本系统严格遵守以下医疗安全原则：

### ⚠️ 安全红线（绝对不做的）

1. **不给出具体用药剂量**
   - 系统不会建议具体的药物剂量
   - 不会开具处方或推荐处方药

2. **不直接开具处方药名**
   - 不会替代医生开具处方
   - 药物相关信息仅供参考

3. **不判断生死预后**
   - 不会预测疾病走向或生存期
   - 不会对生命预后做出判断

4. **不替代医生诊断**
   - 所有分析仅供参考
   - 不作为医疗诊断的依据

### ✅ 系统能够做到的

1. **解读医疗检验指标的临床意义**
   - 解释各项指标代表什么
   - 说明指标异常可能意味着什么

2. **识别异常指标和潜在风险**
   - 标记超出参考范围的指标
   - 提醒可能存在的健康风险

3. **提供健康生活方式建议**
   - 饮食、运动、作息建议
   - 疾病预防的健康指导

4. **推荐针对性检查项目**
   - 建议需要进一步检查的项目
   - 推荐相关科室就诊

5. **协助制定随访计划**
   - 建议复查频率
   - 提醒关注重点指标

6. **整合多学科专家意见**
   - 提供多角度的综合分析
   - 帮助理解各专科意见

### ⚠️ 重要声明

- **本系统所有分析报告仅供参考**
- **不作为医疗诊断依据**
- **所有诊疗决策需咨询专业医生**
- **如有紧急情况，请立即就医**

## 使用限制

### 系统限制

1. **不适用于紧急医疗情况**
   - 如有紧急情况，请立即拨打急救电话或前往医院
   - 本系统无法处理紧急医疗事件

2. **不适用于儿童和孕妇**
   - 系统目前主要针对成人设计
   - 儿童和孕妇的参考标准不同

3. **不适用于严重疾病诊断**
   - 不应用于确诊严重疾病
   - 所有诊断需由专业医生完成

### 数据准确性限制

1. **检查单识别准确性**
   - 依赖于图片质量和清晰度
   - 模糊或不完整的图片可能导致识别错误

2. **参考范围的局限性**
   - 参考范围可能因实验室、设备、方法不同而有所差异
   - 应以就诊医院的参考范围为标准

3. **辐射剂量的估算**
   - 辐射剂量为估算值
   - 实际剂量受设备、参数等多种因素影响
   - 仅作为健康管理参考

### 责任免责

1. **仅供参考**
   - 系统提供的所有信息和分析仅供参考
   - 不构成医疗建议或诊断

2. **专业医疗建议**
   - 所有医疗决策应咨询专业医生
   - 不要仅依赖本系统做出健康决策

3. **数据安全**
   - 用户应自行负责数据备份
   - 系统不承担数据丢失的责任

4. **使用风险**
   - 使用本系统产生的任何后果由用户自行承担
   - 开发者不承担使用本系统导致的任何损失

## 建议的使用方式

### 正确使用

1. **作为健康管理辅助工具**
   - 记录和追踪健康数据
   - 了解健康指标变化趋势

2. **作为就医准备工具**
   - 就诊前整理检查结果
   - 准备向医生咨询的问题

3. **作为健康学习工具**
   - 学习医学检验知识
   - 了解各项指标的临床意义

4. **作为医患沟通辅助**
   - 更好地理解医生的建议
   - 参与自己的健康管理决策

### 不当使用（请避免）

1. **不要仅依赖系统做出诊断**
   - 必须咨询专业医生

2. **不要自行用药或调整用药**
   - 所有用药需遵医嘱

3. **不要延误就医**
   - 症状持续或加重应及时就医

4. **不要过度焦虑**
   - 指标轻微波动是正常的
   - 应由医生判断是否需要治疗

## 紧急情况处理

如果您遇到以下情况，请立即就医或拨打急救电话：

- 胸痛或胸闷
- 呼吸困难
- 严重头痛
- 意识模糊
- 严重外伤
- 过敏反应
- 其他紧急症状

**急救电话：120**
